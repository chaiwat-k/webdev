Body copy

Bootstrap's global default font-size is 14px, with a line-height of 1.428. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their computed line-height (10px by default).

Inline text elements

You can use the <mark> tag to highlight texts.

For indicating blocks of text that have been deleted: unused text, use the <del> tag.

For indicating blocks of text that are no longer relevant: irrelevant use the <s> tag.

For indicating additions to the document use the <ins> tag.

To underline text use the <u> tag.

For emphasizing a snippet to text with a heavier font-weight, use <strong> tag.

For emphasizing a snippet of text with italics, use <em> tag.
